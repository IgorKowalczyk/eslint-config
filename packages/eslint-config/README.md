### 📥 Installation

```bash
npm install --save-dev @igorkowalczyk/eslint-config
yarn add --dev @igorkowalczyk/eslint-config
pnpm install --save-dev @igorkowalczyk/eslint-config
```

### 🔩 Usage

```js
// eslint.config.js
import eslintConfig from "@igorkowalczyk/eslint-config/";

export default [
 ...eslintConfig
];
```
